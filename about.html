The Person Behind the Code

About Me

A little bit about where I come from, what led me to web development, what I'm working on right now, and where I'm headed.

My Background

I grew up in Nairobi, Kenya, in a household where curiosity was encouraged and tinkering with electronics was a Saturday hobby. From an early age I was drawn to understanding how things work — not just using them, but pulling them apart to see the gears inside.

Before I wrote my first line of HTML, I studied Business Administration and spent two years working in logistics. The job was steady but it left me feeling like something was missing. I wanted to build things, not just manage them.

Why Web Development?

The turning point came when I helped a family member set up a basic website for their small business. I expected it to feel like a chore. Instead, I stayed up until 3 AM, completely absorbed. Something clicked. I realised the web is the most accessible creative medium ever invented — the tools are free, the audience is global, and the only real barrier is the willingness to learn.

Since then I have been enrolled in a structured web development bootcamp, spending evenings and weekends deliberately building skills I can actually apply. This portfolio is Week 2's proof of work.

What I'm Learning Right Now

Core Technologies

My current focus is on mastering the fundamentals before touching frameworks. That means writing semantic HTML5, responsive CSS3, and eventually clean vanilla JavaScript. I believe a solid foundation makes everything else easier to pick up later.

Tools & Workflow

I use VS Code as my editor, Git for version control, and GitHub to host and share my projects. I also spend time reading the MDN Web Docs almost daily — it is hands-down the most reliable reference I've found.
